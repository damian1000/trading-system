# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project, please report it privately via GitHub's private vulnerability reporting — the **"Report a vulnerability"** button under the repository's **Security** tab.

Please do not file public issues for security vulnerabilities. You can expect an initial acknowledgement within 14 days.

## Supported Versions

This project follows main-branch development; security fixes are applied to the latest commit on the default branch.
